Project Summary

My cognitive neuroscience research work involves collaboration with researchers at the Department of Neurosurgery at the University of California, San Francisco studying the impairments of cognitive and sensory functions in brain tumor patients. We collect behavioral and intracranial EEG (through placing electrodes directly on the brain) data while patients perform various computerized cognitive tasks that are programmed in MATLAB.

Problem

Research technicians who are responsible for collecting the longitudinal cognitive data from the patients do not have the programming background and the familiarity with MATLAB to be able to quickly run and troubleshoot computerized cognitive testing.

Solution

In an effort to streamline this testing process, I designed and implemented a graphical user interface for running computerized cognitive tests to be used by research technicians. After many iterations of the toolbox based on user feedback, the interface allows testers to:

  • run cognitive task scripts through a button press
  • adjust the parameters of the cognitive task without needing to edit the task script
  • save testing notes
  • allow them to draw on an image of the brain to denote electrode placement
  • pull up previous testing information for a specific subject
  • use checkbox of all the testing procedures to keep track of progress
